List of Requirements

Name

R1 - Add a new contact

Summary

Adds a new contact to the list

Inputs

The first name of the contact. ( the combination of first and last names for a contact must be unique)

The last name of the contact. ( the combination of first and last names for a contact must be unique)

The address of the contact

The email of the contact

Set of phone numbers for the contact

Set of keywords for the contact

Results

If the contact isn't already on the list, it is added. The first and last names are added as keywords for that contact.

If the contact already exists, then it is not added and a warning message is displayed for the user.

Name

R2 - Delete a contact

Summary

Deletes a contact from the list

Inputs

The first name of the contact that is to be deleted.

The last name of the contact that is to be deleted.

Results

The contact is eliminated from the list and its information is no longer available.

Name

R3 - Display contact

Summary

Displays all the information for a contact

Inputs

The first name of the contact to be displayed.

The last name of the contact to be displayed.

Results

All the information for that contact is displayed.

Name

R4 - Modify contact

Summary

Modifies the information for a particular contact. The first and last names can not be edited.

Inputs

The first name of the contact that is to be modified.

The last name of the contact that is to be modified.

New address for that contact.

New email for that contact.

New set of phone numbers for that contact.

New set of keywords for that contact.

Results

The new information for that contact is added, the list of phone numbers and keywords in particular is updated.

Name

R5 - Display the list of contacts.

Summary

A list with the first and last names of all the contacts on the list is displayed

Inputs

None

Results

List with the first and last names of everyone on the list.

Name

R6 - Search for contacts using keywords

Summary

We look for the contacts that have a given keyword in their list

Inputs

Keyword used to search for the contacts.

Results

List with the first and last names of all the contacts that contain that keyword.
